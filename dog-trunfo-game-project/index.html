Bem-vindo ao Super Doguinhos em Confronto!

O que é "Super Doguinhos em Confronto"?

Super Doguinhos em Confronto é um jogo de cartas no estilo Super Trunfo com temática canina!

O jogo foi feito como parte de um projeto para uma diciplina de Sistemas para Internet e é nosso primeiro projeto utilizando JavaScript, então por favor, seja paciente e reporte qualquer erro encontrado!
