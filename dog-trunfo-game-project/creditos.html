Sobre o projeto

Super Doguinhos em Confronto é um jogo de cartas estilo Super Trunfo com temática canina!

O jogo foi feito como parte de um projeto para uma disciplina de Sistemas para Internet e é nosso primeiro projeto utilizando JavaScript, então por favor, seja paciente e reporte qualquer erro encontrado!
